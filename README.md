# tools-sederhana-untuk-enkripsi-dekripsi
Ini buat pembelajaran untuk belajar soal enkripsi dan deskripsi buat pembelajaran 

Cara Menjalankan di Termux / Linux

1. Pastikan Python sudah terinstal:



pkg install python        # Termux
sudo apt install python3  # Linux

2. Jadikan file encryptor.py bisa dijalankan:



chmod +x encryptor.py

3. Jalankan:



./encryptor.py

Atau:

python3 encryptor.py
